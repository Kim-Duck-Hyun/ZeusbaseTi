App Icons Module

Description

254 vector-based, commonly used icons in one true-type font, featuring icons for web application, editor, video player and social media. Icons can be scaled and colored by font attributes without loss of quality at any size.

Installation

  1. Unzip the downloaded package. Copy the AppIcons.ttf and icons.js files into your project.
  2. Follow the instructions from Appcelerator to install icon font AppIcons.ttf (size: 64KB).
  3. Include icons.js module into your project.

Using the App Icons Module

To use the icon simply put icons.icon_name in places where string is expected. See the example codes included in the package for inspiration.
