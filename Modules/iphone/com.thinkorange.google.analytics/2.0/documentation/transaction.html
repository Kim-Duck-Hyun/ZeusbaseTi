google_analytics Module

Transaction

A simple class to hold transaction data. You can obtain a reference to a transaction object using this:

var Transaction = GoogleAnalytics.createTransaction({
  transactionID: '1'
});

The required fields are:

  • transactionID: The transaction ID. Required

Properties

Transaction.affiliation

The transaction affiliation.

Transaction.revenueMicros | Transaction.setRevenueMicros(int)

Revenue in micros (millionths of a currency unit). Note that this must be set manually because it is not updated when items are added.

Transaction.taxMicros | Transaction.setTaxMicros(int)

Tax in micros (millionths of a currency unit). Note that this must be set manually because it is not updated when items are added.

Transaction.shippingMicros | Transaction.setShippingMicros(int)

Shipping cost in micros (millionths of a currency unit). Note that this must be set manually because it is not updated when items are added.

Methods

Transaction.addItem({...})

Add an item to the transaction. If an item with the same SKU already exists in the transaction, that item will be replaced with this one. It accepts the following parameters:

  • productCode [required]: The item product code, required.
  • productName [required]: The item product name
  • productCategory [required]: The item product category
  • priceMicros [required]: The item price, in micros (millionths of a currency unit)
  • quantity [required]: The item quantity, required.

Submitting a transaction

Example:

var Tracker = ...
var Transaction = GoogleAnalytics.createTransaction({
  transactionID: '1',
  affiliation: 'In-App Store',
  taxMicros: 0.17 * 1000000,
  shippingMicros: 0,
  revenueMicros: 2.16 * 1000000
});

Transaction.addItem({
  productCode: 'SKU_789',
  productName: "Level Pack: Space",
  productCategory: "Game expansions",
  priceMicros: 1.99 * 1000000,
  quantity: 1
});

Tracker.trackTransaction(Transaction);
