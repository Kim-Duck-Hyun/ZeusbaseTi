How to load the gallery module

  1. Copy the “lib” and “common” folder (from the “source” folder) inside you project “Resources”  folder (old or new)
  2. Copy all the strings from “strings.xml” file (from the “source->i18n->en” folder) into your “strings.xml” (or if you don’t have that file-folder copy the folder “i18n” into your project next to the “Resources” folder)
  3. Now the files are in place. To load the modules add the following code somewhere in top of app.js or some other file
//################# MODULES ##################
var VARS =require('common/globals'); //Here we store all global variables / settings
var Gallery=require('lib/galleryModule'); //Here are the functions to create the gallery


This is the folder structure you should end with.

How to use the gallery

  1. First create a window and tab where you want the gallery to be opened in. You will pass later this objects as parameters.  If you are new to Titanium check the
    “example->Reources->app.js” to see how to create that window and the tab.
  2. To create the gallery (flicr,facebook,picasa or nextgen) this is the structure
 Gallery.createGalleryList({
      win:galleryWindow,         //The window to display in [Required]
      title:"NextGen Gallery",   //Title of the window [Required]
      type:VARS._NEXT_GEN,       //The type of the gallery [Required] [_NEXT_GEN,_FB_GALLERY,_PICASA,_FLICKR]
      tab:tab1,                  //The tab that gallery will open its sub views [Required]
      openType:1,                //For future modifications of the gallery [Required]
     
      id:1,                      //Id of the NextGenAlbum [Required if type=_NEXT_GEN]
      url:VARS._NextGen_Address, //Adress of your blog [Required if type=_NEXT_GEN]

      page:VARS._FBPageID,       //Facebook Page id  [Required if type=_FB_GALLERY]
   
      userID:VARS._PicasaUserId,  //Picasa User id [Required if type=_PICASA]

      appKEY:VARS._FlickrAppKEY,  //Flickr app key [Required if type=_FLICKR]
      userID:VARS._FlickrUserID,  //Flickr User id [Required if type=_FLICKR]
                          });

After this call the gallery will be in the wiindow you passed as parameter. in this example galleryWindow.

Then you should open that window in the tab you passed as parameter.

tab1.open(galleryWindow);

 

Configure gallery

Now you will configure the gallery to use your gallery insted of the default.
This will be very easy
 

  1. Open "common->globals.js"
  2. In the top you will notice "USER DATA"
  3. Change this values to suit yor needs
  4. Build the project, You will see your photos.

*Remember to change the Flickr App Key with your own

Changes colors

  1. Once again go into "common->globals.js"
  2. Here you will notice "GLOBAL COLORS"
  3. Change the values to suit your needs.

Facebook Share Settings

To set up the Facebook Share just add the following code in your app.js

//############### SHARE SETTINGS #####################
Ti.Facebook.appid = "134793934930"; //Change this with your own Facebook app id
Ti.Facebook.permissions = ['publish_stream', 'read_stream'];
