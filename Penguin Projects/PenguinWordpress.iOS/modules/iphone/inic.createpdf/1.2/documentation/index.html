createpdf Module

Description

This module allows you to create PDF from the JSON string. You need to create One JSON string from your Javascript code and then pass it to createPDF function as a parameter. You can find the JSON string Skeleton in example. Generated PDF is saved at application data directory.

Updates in version 1.1 :

  • Remote Image support to be set as background of PDF file to be generated.

Updates in version 1.2:

  • Add style to the text element (for ex. font size, font color, font style[i.e. normal,bold], width, height, left margin etc.)
  • Define desired width, height, default margins of your PDF file.
  • You can render remote Images,images from application resource directory as well from application data directory.
  • You can define desired width, height and left margin of your image element.

see example for the JSON string skeleton.

Accessing the createpdf Module

To access this module from JavaScript, you would do the following:

var createpdf = require("inic.createPDF");

The createpdf variable is a reference to the Module object.

Reference

var view = createpdf.createView();

Function

view.createPDF(pdfname,pdfbgimg,pdfdata,pdfsize);

This function has four string argument(pdfname,pdfbgimg,pdfdata,pdfsize).

pdfname - This parameter contains name of the pdf to be generated.

For Ex: var pdfname = "IndiaNIC.pdf";

pdfbgimg - This parameter contains background image of the pdf file to be generated. Preferable Dimension are 320X480 and 640X960 for iPhone, 768X1024 for iPad

For Ex: Case 1: Local Image (Note: Must resides in Resource Directory/Application Data Directory) var pdfbgimg = "pdfbg.png";

Case 2: Remote Image (Online URL of image) var pdfbgimg = "http://www.domainname.com/bgimage.jpg";

pdfdata - Third parameter contains JSON string data. You can find example of this JSON string in "app.'s" in the example folder.

pdfsize- Fourth parameter contains JSON string, which define pdf dimensions like width,height and default margins(i.e. left margin ,top margin etc.).You can find example of this JSON string in "app.'s" in the example folder.
